个人征信报告（二代格式）

│

│

├─ 0. 表头

│    ├─ 被查询者姓名

│    ├─ 被查询这证件类型

│    ├─ 被查询这证件号码

│    ├─ 查询机构

│    ├─ 查询原因

│    ├─ 异议信息提示

│

├─ 1. 个人基本信息

│   ├─ 身份信息：性别/出生日期/学历/学位/国籍/电子邮箱/民族/婚姻状况/通讯地址/户籍地址/手机号码

│   ├─ 配偶信息：姓名/证件类型/证件号码/工作单位/联系电话

│   ├─ 居住信息：居住住址/住宅电话/居住状况/信息更新日期

│   └─ 职业信息：工作单位/单位性质/单位地址/单位电话/职业/行业/植物/支撑/进入单位年份/信息更新日期

│

├─ 2. 信息概要

│   ├─ 信贷交易信息提示：贷款/信用卡笔数、授信总额/已用额度、最近6个月平均使用额度

│   ├─ 信贷交易授信及负债信息概要：非循环贷账户信息汇总/循环贷账户二信息汇总/贷记卡账户信息汇总/相关还款责任信息汇总

│   └─ 查询记录概要：最近1个月内的查询机构数/最近一个月的查询次数/最近2年内的查询次数

│

├─ 3. 信贷交易信息明细

│   ├─ 非循环贷账户（以单个账户为例。可能出现多个账户，但结构相同）：管理机构/账户标识/开立日期/到期日期/借款金额/账户币种/业务种类/担保方式/还款期数/还款频率/还款方式/共同借款标志/账户状态/账户关闭日期

│   ├─ 循环贷账户一（以单个账户为例。可能出现多个账户，但结构相同）：管理机构/账户标识/开立日期/到期日期/借款金额/账户币种/业务种类/担保方式/还款期数/还款频率/还款方式/共同借款标志/账户状态/账户关闭日期

│   ├─ 循环贷账户二（以单个账户为例。可能出现多个账户，但结构相同）：管理机构/账户标识/开立日期/到期日期/借款金额/账户币种/业务种类/担保方式/还款期数/还款频率/还款方式/共同借款标志/账户状态/账户关闭日期

│   ├─ 贷记卡账户（以单个账户为例。可能出现多个账户，但结构相同）：发卡机构/账户表示/开立日期/账户授信额度/共享授信额度/币种/业务种类/担保方式/账户状态/余额/已用额度/未出单的大额专项分期余额/剩余分期期数/最近6个月平均使用额度/最大使用额度/账单日/本月应还款/本月实还款/最近一次还款日期/当前逾期期数/当前逾期总额 

│   ├─ 相关还款责任信息：

│   └─ 授信协议信息（以单个协议为例。可能出现多个账户，但结构相同）：管理机构/授信协议表示/生效日期/到期日期/授信额度用途/授信额度/授信限额/授信限额编号/已用额度/币种

│

│

├─ 4. 查询记录

&nbsp;    ├─ 机构查询记录明细：编号/查询日期/查询机构/查询原因

&nbsp;    └─ 本人查询记录明细：编号/查询日期/查询机构/查询原因
